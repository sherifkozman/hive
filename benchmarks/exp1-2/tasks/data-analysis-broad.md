# Task: data-analysis / BROAD

Using `eval/fixtures/sales.csv`, produce a complete analysis report for a
non-technical VP of Sales. You may write and run Python; the deliverable is a
Markdown report containing:

1. Data quality assessment (missing values, anomalies, duplicates — and how you
   handled them).
2. Revenue overview: totals, trends over time, by region and category.
3. Three non-obvious insights (supported by numbers, not just aggregates).
4. Discounting analysis: is discounting working? (relationship between discount
   and revenue/volume).
5. Three concrete, prioritized recommendations with expected impact.

Numbers must be correct and reproducible; state any assumptions. Tables over
prose where it helps. No charts required (text/tables only).
